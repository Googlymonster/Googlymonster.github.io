About me

My name is Kevin Nguyen, I graduated from Texas Tech University with a degree in Chemical Engineering in 2013. I worked as a field engineer from 2013 to 2017 before changing career paths into an operations engineer in 2017. From 2017 to 2018 I worked as an operations engineer before an opportunity arose to be a technical services engineer. Currently I'm working as a Compliance Data Specialist doing ETL data work.

I have always loved science and engineering since I was young. I try to find solutions to hard problems my whole life. During the years I have continue to learn to expand my skillset as an engineer, which lead me to start coding. Being that my father is a software engineer it help me fast track my knowledge in this skill. I consider myself a self-starter that is motivated to continue learning all aspects of engineering.

My current web backend stack is based around Python, with web development experience using Django/Flask. Frontend experience using javascript involves React/Vue with vanilla javascript using DOM manipulation. I chose python because I started to use it for ETL data manipulation, along with SQL and VBA as well. Using Jupyter Notebook to do my scripting and apply some data science as well. Utilizing Technologies like Docker, Redis, AWS, and etc I've been able to create various projects.
